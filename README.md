
# Case de testes do processo seletivo da Webmotors

# Execução

Tendo Ruby na versão 2.7 e a gem bundle instalada, executar:

```
cd testes_webmotors/
bundle install
cucumber features/
```

Após isso, acessar o relatório na raiz do projeto chamado "relatorio.html"
